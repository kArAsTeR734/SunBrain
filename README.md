# Запуск проекта
Для запуска проекта установите все зависимости через команду 

### npm i
После установки зависимостей напишите команду
### npm run dev.

В случае отсутствия пакетного менеджера npm на устройстве - установите Node.js с официального сайта последней поддерживаемой версии, npm идёт вместе с ним.
